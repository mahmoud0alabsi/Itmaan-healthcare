## Getting Started

### Prerequisites

- Flutter SDK
- Android Studio / VS Code
- Git

### Installation

1. Clone the repository
```bash
git clone [repository-url]
```

2. Navigate to project directory
```bash
cd itmaan
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
